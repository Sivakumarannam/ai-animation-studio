# Project Vision

This is **NOT** a manual animation editor.

The goal is to build a **fully autonomous AI YouTube Automation Platform** using only **free and open-source technologies**.

The user should only have three manual actions:

* Start
* Pause
* Delete

After pressing **Start** once, the platform should continue automatically every day until paused or deleted.

The platform will eventually:

* Discover trending topics
* Research topics
* Perform fact verification
* Build a long-term knowledge base (RAG)
* Generate stories
* Maintain world and character consistency
* Generate scripts
* Generate images
* Generate backgrounds
* Generate props
* Generate animations
* Generate voices
* Generate subtitles
* Generate music
* Render videos
* Generate thumbnails
* Generate YouTube SEO
* Publish automatically
* Learn continuously from performance

Only free/open-source tools are allowed.
